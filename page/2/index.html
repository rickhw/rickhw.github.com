輕鬆聊：系統測試 (SVT) 的三兩事 2018/03/18 23:33:00

在 Stages in Software Testing 一文有提到 系統驗證測試 (System Verification Test) 的基本概念，基本的想法就是：

  • 功能驗證 (FVT)：假設功能還沒好，目的是確認商業功能正確性，像是邏輯完整性、資料正確性、驗證與錯誤處理 … 等，強調功能本身的 內聚力。
