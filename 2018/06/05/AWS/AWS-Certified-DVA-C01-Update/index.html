AWS Certified DVA-C01 Update

新版的 AWS Certified Developer (Release June 2018)，考試代號 (DVA-C01)，範圍果然改了，多不少東西。

考試基本資訊

  • AWS Certified Developer (Release June 2018)
  • DVA-C01 Exam Guide

Whitepapers 版本差異

  • Cloud 核心概念
    • AWS Security Best Practices
    • AWS Well-Architected Framework
    • Architecting for the Cloud AWS Best Practices
  • 新的部分涵蓋 Serverless、Microservice、Containerized、CI/CD 等 …
    • Microservices on AWS, September 2017
    • Serverless Architectures with AWS Lambda, November 2017
    • Optimizing Enterprise Economics with Serverless Architectures, October 2017
    • Running Containerized Microservices on AWS, November 2017
    • Practicing Continuous Integration and Continuous Delivery on AWS Accelerating Software Delivery with DevOps, June 2017
    • Blue/Green Deployments on AWS, August 2016

範圍

下面從 Exam Guide 截下來的圖：

考試方式

  • 答題方式：單選，複選
  • 題目數量：55 -> 65， 多十題
  • 考試時間：80min -> 130min
  • 費用：150USD 不變
  • 計分方式：Exam Guide 提到分數範圍是 100-1000，720 分過關。
    • 整體分數過關即可，不考慮單一個 Domain 是否一定要過關

問題

舊版的還能考？

到 2018/11/29 之前都還可以考，已經在準備的人可以繼續準備

新版的什麼時候可以考？

現在已經可以考。
