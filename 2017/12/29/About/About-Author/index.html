關於作者

罐頭式介紹

其實每次演講都不知道要怎麼介紹自己，因為做過很多事，通常都寫 打雜的。底下整理各種場合需要用的自我介紹，需要時候填資料比較方便 XD

官方說法

底下是官方說法，用於正式場合，像是代表公司的演講、公開的聚會 … 等。

自許為 軟體開發者 (Software Developer)，有二十多年軟體開發經驗，寫過 PHP、Java、C、Python、COBOL … 等程式語言。工作角色經歷 Software Developer、SQA Manager、Operation Manager 等職務，開發過大型 Eclipse Plugins 架構、大型自動化測試架構、維運大型電商系統，擅長軟體工程、軟體測試、系統架構。這幾年專注在經營管理、AWS / GCP、分散式架構、產品系統維運、DevOps / SRE 等領域，並活躍於各大社群的活動、演講。目前任職於台灣著名電商龍頭資深經理、技術委員會，負責管理 MIS / Infra / Cloud、產品系統維運、基礎服務架構、分散式架構開發 … 等工作，同時也是內部技術推廣者，包含 AWS / SRE / 持續交付 等。另外也是台北 TGO 鯤鵬會會員，SRE Taiwan 社群義工。

社群介紹

用在社群的演講、非官方、半正式，不想太有距離感的寫法，多一點個人的色彩。

想拯救世界的 軟體開發者 (Software Developer)、音樂愛好者，工作角色經歷 DevOps 三個圈圈職務: Dev / QA / Ops。專長為系統架構、軟體開發、軟體測試、大型系統維運，這幾年專注在經營管理、AWS / GCP、分散式架構、維運、DevOps / SRE 等領域，並活躍於各大社群的活動、演講，著有技術部落格：Complete Think。工作之外喜歡科幻小說、金庸武俠、經典文學、哲學，也是業餘音樂愛好者，涉略涵蓋 吉他、鍵盤、編曲、教學，著有音樂部落格：喝咖啡聊音樂 以及 電子書 出版。
